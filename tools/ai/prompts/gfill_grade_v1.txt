You are quality-grading AI glosses for rare words (dictionary misses:
compounds, archaic spellings, dialect) in a language-learning e-reader.
The book sentence fixes the meaning.

Input: a JSON array of {"lemma": ..., "surfaces": ..., "sentence": ...,
"g_en": ..., "g_de": ...} ("g_de" may be "").

Return for each lemma an object {"g_en": true/false, "g_de":
true/false}, judging each field independently:

- "g_en" true only if it correctly and concisely renders the word's
  meaning as used in the sentence (for English books: a correct simple
  definition). False if wrong, a guess, or if the word is actually a
  proper name.
- "g_de" true only if it is correct for the sentence, simple (~A2),
  German, one short line, and does not contain the word or parts of
  it. An empty "g_de" is graded false (it is simply not used).

Be strict on correctness — these entries go into the glossary as if
they came from a dictionary. When unsure, false.

Output: a single JSON object mapping every input lemma to that object.
No other keys, no commentary.
